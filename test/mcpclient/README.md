# mcpclient 测试

本目录包含 `pkg/mcpclient` 包的测试示例。

## 运行测试

```bash
# 进入测试目录
cd test/mcpclient

# 运行测试
go run main.go
```

## 测试内容

1. **配置加载测试** - 从文件加载客户端配置
2. **配置验证测试** - 测试 nil 配置、空服务器列表、无效配置的验证
3. **选项模式测试** - 测试使用选项模式创建客户端
4. **自定义日志记录器测试** - 测试使用自定义日志记录器
5. **连接测试** - 测试连接到服务器（需要服务器运行）
6. **命令执行测试** - 测试执行命令并解析结果

## 注意事项

- 连接测试需要服务器正在运行
- 如果服务器未运行，连接测试会失败，这是正常的
- 测试使用 `../client_config.json` 配置文件
